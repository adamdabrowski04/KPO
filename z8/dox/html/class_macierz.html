Opis szczegółowy

template<class TYP>
class Macierz< TYP >

Modeluje macierz.

Szabon klasy modeluje pojęcie macierzy kawadratowj o określonej wielkości(parametr szablonu) dla obiektów dwolnego typu. Głównym elementem klasy jest szablony klasy Wektro którego parametrem jest drugi szablon klasy wektor
