Opis szczegółowy

template<class TYP>
class Wektor< TYP >

Modeluje wektor.

Szabon klasy modeluje pojęcie wekor o określonej długośc (parametr szablonu) dla obiektów dwolnego typu. Głównym elementem klasy jest wskażnik do obszaru pamięci zawierającego dane
